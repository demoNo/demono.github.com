Android 运行时权限

发表于 2017-01-23 | 分类于 Android | | 阅读次数

在Android6.0（API-23）以上且 targetSdkVersion 是23时申请危险权限需要动态申请。

如果设备运行的是 Android 5.1（API 级别 22）或更低版本，并且应用的 targetSdkVersion 是 22 或更低版本则不需要申请权限。

运行时权限的几个基本方法

API 作用
checkSelfPermission() 判断权限是否具有某项权限
requestPermissions() 申请权限
onRequestPermissionsResult() 申请权限回调方法
shouldShowRequestPermissionRationale() 是否要提示用户申请该权限的缘由
shouldShowRequestPermissionRationale 方法只有当用户第一次拒绝之后再请求时会返回true 如果用户勾选不再询问之后会返回false

如下事例

序号 用户是否授予权限 shouldShowRationale() 返回 是否勾选“不再询问”
1 否 false -
2 否 true 否
3 否 true 否
…………i 否 true 是
i + 1 -false - -
